# Board Update — One Pager

## The Page
- **Headline** — what changed since last check-in.  
- **Scoreboard** — 5 numbers that matter.  
- **Risks / Mitigations** — truth without theatre.  
- **Asks** — decisions, introductions, support.

### Rules
- Fit on one page or you don’t understand it yet.  
- Attach the appendix only if requested.

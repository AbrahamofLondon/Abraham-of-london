If you have tried three fixes and nothing changed, stop relabelling the issue.

That is not bad luck. That is structure protecting itself. The authority path is broken, hidden, or politically unsafe — and every surface-level fix routes around the real issue instead of through it.

This system is built for exactly this moment. The point where you have tried enough to know the problem is not what it appears to be. Six questions. It tells you which failure mode you are in.

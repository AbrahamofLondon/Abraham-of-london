The king chose stone because stone outlasts memory. Napoleon chose realist painting because realism looked like truth. The empire chose photography because photography had the aura of mechanical objectivity. The state chose poster art because posters could be everywhere simultaneously.

Facebook chose engagement optimisation because engagement is measurable, monetisable, and compounds.

The tool is never neutral. The tool is always the choice of whoever built it, for the purpose they built it for. The algorithm was built to maximise engagement. It is doing exactly what it was built to do. The question is not whether the algorithm is broken. The question is whether the optimisation target was the right one.

The same question applies to every system you design, every metric you choose, every incentive you set. What is your system optimising for? And is that what you would have chosen if you had been asked directly, before the system started running?

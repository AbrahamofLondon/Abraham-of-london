This is not unusual. It is the default condition of born-digital work.

Format obsolescence alone — software that requires hardware that is gone, file formats that no application can open — has rendered unreadable entire categories of digital output from the 1980s and 1990s.

Not deleted. Unreadable. The distinction matters: the data exists but cannot be accessed.

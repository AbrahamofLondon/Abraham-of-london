# Council Table Agenda

## A Sovereign Framework for Council Meetings

### Overview
The Council Table Agenda is a structured framework for gatherings where decisions matter. Unlike conventional meeting templates that focus on logistics, this framework centers on sovereignty, clarity, and accountability.

### Part I: Pre-Council Preparation

**Strategic Context Review**
- Review previous council decisions
- Assess current strategic position
- Identify emerging threats and opportunities

**Decision Inventory**
- List all items requiring decision
- Classify by urgency and importance
- Assign preparation owners

**Required Readings**
- Distribute materials 7 days prior
- Confirm receipt by all members
- Document questions in advance

**Participant Readiness Assessment**
- Has each member completed preparation?
- Are any members unable to attend?
- Is quorum achievable?

### Part II: The Agenda Structure

**1. Opening of Council (15 min)**
- Principles invocation
- [Sovereignty](/vault/lexicon/sovereignty) check
- Attendance confirmation
- Previous minutes approval

**2. Strategic Review (30 min)**
- Previous decisions audit
- Environmental scan
- Risk assessment
- Opportunity review

**3. Decision Items (60–90 min)**
- Item presentation
- Sovereign deliberation
- [Clarity](/vault/lexicon/clarity) check
- Formal decision
- Documentation assignment

**4. Council Close (15 min)**
- Summary of commitments
- Next council preparation
- Closing of council
- Final principles affirmation

### Part III: Post-Council Documentation

**Decision Registry Update**
- Record all decisions with rationale
- Assign action items
- Set review dates

**Action Item Tracking**
- Owner
- Deadline
- Dependencies
- Success criteria

**Council Minutes (Sovereign Format)**
- Decisions only (not discussion)
- Rationale summary
- Dissenting views (if any)
- Commitments

### Part IV: Decision Matrix

| Decision Type | Threshold | Timeline | Documentation |
|---------------|-----------|----------|---------------|
| Strategic | Consensus | Multi-session | Full rationale |
| Tactical | Majority | Single session | Summary |
| Operational | Individual | Immediate | Brief note |
| Emergency | Any | As needed | Retroactive |

### Usage Protocol

1. Distribute agenda 7 days prior to council
2. Each member completes readiness assessment
3. Bring printed copy to council
4. Document decisions directly on agenda
5. Archive in institutional ledger within 48 hours
6. Review adherence at next council opening

### Sample Decision Log Entry

```
Date: [DATE]
Decision: [DECISION]
Proposed by: [NAME]
Seconded by: [NAME]
Rationale: [SUMMARY]
Alternatives considered: [LIST]
Vote: [FOR/AGAINST/ABSTAIN]
Action items: [LIST]
Review date: [DATE]
```

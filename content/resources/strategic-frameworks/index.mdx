# Strategic Frameworks

**This is the library.**
A disciplined collection of decision systems designed for leaders operating under scrutiny — where clarity must survive pressure.

## What this is
A set of **board-grade frameworks** with:
- **Decision rights** (who owns what)
- **Inputs** (what must be true before deciding)
- **Outputs** (what the board expects to see)
- **Failure modes** (how teams misuse strategy)

## Who this is for
- Founders carrying irreversible decisions
- Directors governing legitimacy and risk
- Operators building cadence and accountability
- Households building order that survives stress

## How to use this library
1. Start with **v4** (canonical current system)
2. Apply the **Decision Hygiene** workflow
3. Produce an **artifact** (memo, decision log, cadence plan)
4. Review and iterate on cadence

## Quick-start: Decision Hygiene (Board Standard)

### Inputs (required)
- Objective (one sentence)
- Constraints (cash, time, people, compliance)
- Options (max 3)
- Risks (top 5, with mitigation)
- Authority (who decides, who advises)

### Process (45–90 minutes)
- Frame the decision (what is truly being decided?)
- Score options (impact, risk, reversibility, time-to-value)
- Decide (record owner + rationale)
- Commit to next actions (with dates)

### Outputs (what the board expects)
- 1-page decision memo
- Decision log entry
- Risk register delta
- KPI impact statement
- Governance implication note

## Standards (non-negotiable)
Strategy fails when it becomes performance. This library is built to prevent that.

- **If it cannot be executed, it is not strategy.**
- **If it cannot be audited, it is not governance.**
- **If it cannot be explained in plain speech, it is not clarity.**

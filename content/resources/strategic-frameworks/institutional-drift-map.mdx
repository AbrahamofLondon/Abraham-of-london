## 1. Executive Summary

Institutions do not collapse suddenly.

They drift.

Slowly. Quietly. Incrementally.

```txt
Misalignment accumulates  
Standards weaken  
Authority blurs  
Reality diverges from intent  
```

The Institutional Drift Map identifies:

```txt
Where drift is occurring  
What stage the organisation is in  
How severe the deviation is  
What must be corrected  
```

It converts drift from:

```txt
Invisible → Measurable → Actionable
```

---

## 2. Strategic Context

Every institution begins with:

```txt
Clarity of purpose  
Strength of structure  
Discipline of execution  
```

Over time, four pressures emerge:

```txt
Time  
Success  
Complexity  
Human compromise  
```

These produce drift.

Unchecked drift becomes decay.
Unchecked decay becomes collapse.

---

## 3. Core Model — Drift Cycle

The framework operates across **4 stages**:

```txt
ORDER → DRIFT → DECAY → COLLAPSE
```

---

### 3.1 ORDER

```txt
Clear identity  
Aligned leadership  
Strong governance  
Consistent execution  
```

---

### 3.2 DRIFT

```txt
Small deviations tolerated  
Standards inconsistently applied  
Early misalignment appears  
```

---

### 3.3 DECAY

```txt
Structural weaknesses visible  
Execution inconsistent  
Leadership divergence  
```

---

### 3.4 COLLAPSE

```txt
System failure  
Loss of control  
Reactive decision-making  
```

---

## 8. Operating Logic

Drift is not random.

It follows a pattern:

```txt
Tolerance → Normalisation → Reinforcement → Entrenchment
```

What is tolerated once becomes:

```txt
Accepted → Expected → Protected
```

This is how institutions decay without noticing.

---

## 9. Drift Detection Process

### Step 1 — Identify signals

Capture recurring failures or inconsistencies

### Step 2 — Map to domain

```txt
Identity / Governance / Culture / Systems / Execution
```

### Step 3 — Classify stage

```txt
Order / Drift / Decay / Collapse
```

### Step 4 — Measure severity

Apply LOW → CRITICAL

### Step 5 — Prioritise correction

Focus on highest severity + highest impact domain

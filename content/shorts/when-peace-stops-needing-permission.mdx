You used to need conditions to be at peace.

If only they would apologise.
If only the situation would resolve.
If only everything was in order.

But you've noticed something changing.

Peace is arriving without permission.

Not because everything is fixed.
Because you're no longer waiting for everything to be fixed
before you're allowed to be okay.

---

Peace that depends on circumstances
is not peace. It's a ceasefire.

And real peace doesn't need conditions.
It just needs to be received.

This week, receive peace without explaining it.

Don't justify why you're okay.
Don't wait for the other shoe to drop.
Just let yourself be at peace.

You don't need permission to be well.
You only need to stop withholding it from yourself.

Something is changing.

You've stopped lying to yourself.

Not about big things at first.
Just small things.

The excuse you no longer believe.
The pattern you can no longer ignore.
The truth you can no longer avoid.

---

Self-honesty is not a single event.
It's a practice.

And every time you choose it,
you become a little more free.

This week, write one sentence you have avoided.

One truth about yourself.
One pattern you've been pretending isn't there.
One thing you know you need to change.

Don't share it. Don't analyse it.
Just write it.

The truth does not need to be announced to set you free.
It only needs to be acknowledged.

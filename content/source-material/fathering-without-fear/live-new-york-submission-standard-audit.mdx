# Fathering Without Fear — Major-Agent Acquisition Readiness Protocol

## Version

live-acquisition-readiness-protocol-post-ch01-ch13-ch22

## Purpose

This document exists to make rejection less likely.

It is not a praise document.
It is not a confidence document.
It is not a motivational document.
It is not a literary mood board.

It is an acquisition-readiness protocol for preparing *Fathering Without Fear* for serious New York / London literary-agent submission and, through the right agent, senior-editor acquisition consideration.

No document can guarantee acceptance. What can be built is the strongest possible submission machine: a manuscript with no obvious craft weakness, no avoidable legal exposure, no weak opening, no collapsed middle, no unearned thesis, no child-protection failure, no unclear market position, and no careless agent targeting.

The standard is not "good enough to send."

The standard is:

```md
Would a serious literary agent risk their reputation, time, and submission capital on this manuscript?
```

If the answer is not clearly yes, the manuscript is not ready.

---

# Section 1 — The Acquisition Reality

## 1. The target is not "publication" first

The immediate target is:

```md
Representation by the right literary agent.
```

The second target is:

```md
Agent-led submission to editors capable of acquiring ambitious literary memoir.
```

The third target is:

```md
An editor who sees the manuscript as literary, urgent, defensible, and publishable at a serious level.
```

The fourth target is:

```md
Publication with enough editorial, publicity, positioning, and reader trust to compete beyond niche memoir.
```

Do not confuse these stages.

A manuscript can be powerful and still fail if sent to the wrong agents, positioned in the wrong category, over-explained in the query, legally exposed, or structurally uneven in the sample pages.

# Section 2 — Submission Definition

For this book, submission readiness means all of the following are true:

1. The full manuscript reads as one literary work, not a collection of strong chapters.
2. The first ten pages create trust immediately.
3. The first fifty pages make the agent want the whole book.
4. The middle does not sag, explain, litigate, sermonise, or repeat.
5. The legally sensitive material is transformed into literary pressure, not argued as a case.
6. Damisi is never used as evidence.
7. The author is accountable without self-erasure.
8. The faith material is credible to a sophisticated secular reader without being diluted for that reader.
9. The lineage material feels like inheritance under pressure, not family prestige.
10. The ending lands as accumulated presence, not resolution.
11. The query and proposal position the book without reducing it.
12. The agent list is built from deal evidence and category fit, not prestige guessing.

# Section 3 — Non-Negotiable Editorial Gates

## Gate 1 — First Ten Pages

The first ten pages must pass the bored-agent test.

The reader must not need goodwill.

The reader must not need context.

The reader must not need sympathy for the author.

The reader must not need to agree with the author.

The reader must only need to keep reading.

### Ch.1 current status

Ch.1 is provisionally cleared.

Current opening image:

```md
Hounslow bedroom.
Bed.
Phone.
Court voice call.
Unnotified hearing.
Judge about to join.
Abraham stays on the line.
```

Do not revise Ch.1 again unless the whole-manuscript read reveals a structural failure.

Ch.1 must remain short, controlled, and unannounced.

Final line:

```md
Abraham stayed on the line.
```

This is the correct opening close.

## Gate 2 — Scene Before Explanation

Every chapter must be tested sentence by sentence:

```md
Is this sentence inside the scene, or is it explaining the scene's meaning?
```

If the answer is explanation, cut unless the chapter collapses without it.

A chapter that explains itself is not submission-ready.

## Gate 3 — No Thesis Announcement

The governing questions must not appear as visible thesis statements.

The reader must feel the questions before the narrator names anything.

Forbidden in chapter body unless structurally unavoidable:

```md
What does a father owe...
This is what fathering without fear means...
The pattern was...
The lesson was...
The system was...
This chapter is about...
```

## Gate 4 — No Legal File

Legal material is allowed only as pressure.

It must not become:

* witness statement;
* position statement;
* chronology;
* skeleton argument;
* complaint;
* prosecution;
* evidence bundle;
* procedural explainer.

Test:

```md
Could this sentence appear naturally in a court filing?
```

If yes, it does not belong in memoir prose.

## Gate 5 — No Sermon

Faith must appear as:

* prayer;
* inherited language;
* misreading;
* correction;
* silence;
* cost;
* covenant;
* endurance;
* unresolved honesty.

Faith must not appear as:

* instruction;
* doctrine dump;
* moral lesson;
* prosperity frame;
* theological proof;
* reader-directed sermon.

Test:

```md
Is the sentence addressed to God, memory, or the wound — or is it addressed to an imagined congregation?
```

Only the first belongs.

## Gate 6 — No Ancestry Brochure

Lineage must appear as pressure, not prestige.

Allowed:

* road;
* name;
* trade;
* cloth;
* work;
* loss;
* inherited burden;
* unresolved memory;
* oral tradition held carefully.

Forbidden:

* family greatness display;
* catalogue of importance;
* lineage as self-justification;
* ancestry as brand positioning.

## Gate 7 — No Revenge Document

Every person must remain human.

No one may be reduced to:

* villain;
* diagnosis;
* symbol;
* obstacle;
* proof;
* legal function;
* spiritual caricature;
* moral lesson.

This is especially important for:

* Serena;
* Damisi's mother;
* court actors;
* Cafcass;
* family members;
* the dead.

## Gate 8 — Child-Protection Absolute

Damisi is not the book's argument.

Every Damisi sentence must pass this test:

```md
Would this sentence still belong if no legal proceedings existed?
```

If no, cut it.

Damisi may appear as:

* boy;
* voice;
* play;
* joke;
* preference;
* imagination;
* body in ordinary childhood;
* son loved by father.

Damisi may not appear as:

* evidence;
* proof;
* symptom;
* legal support;
* moral leverage;
* damaged child claim;
* argument against anyone.

## Gate 9 — Legal and Privacy Red-Line

The following remain prohibited in public prose unless specialist legal review explicitly clears them:

* sealed orders;
* court transcripts;
* named assessments;
* allegation detail;
* private child-sensitive statements;
* medical or psychological claims about Damisi;
* adversarial mother/respondent framing;
* Cafcass rival-opening material;
* detailed Ch.16 protected material;
* plane-crash detail in Ch.11 unless author explicitly reopens privacy decision.

No new prose.

2. Explanation density.
3. Maximum truth within legal/privacy limits.
4. Exact dates if confirmed.

# Section 6 — Market and Submission Package Gates

A strong manuscript is not enough.

The package must also be acquisition-ready.

## Gate A — Category Positioning

The book should not be pitched as:

* legal memoir;
* family court exposé;
* trauma memoir;
* Christian testimony;
* father's rights manifesto;
* immigration hardship memoir;
* Nigerian family history.

Those are ingredients, not the category.

Working category:

```md
Literary memoir about fatherhood, inheritance, faith, and the cost of refusing disappearance under institutional and intimate pressure.
```

## Gate B — Comps

Build a comp matrix with three types of comparables.

### Literary craft comps

Books that justify the prose ambition.

Possible families of comparison:

* high-control literary memoir;
* inheritance memoir;
* grief memoir;
* spiritual memoir without sermon;
* father/son or parent/child literary nonfiction.

### Market-position comps

Books that show where a bookseller, agent, or editor understands the shelf.

### Contrast comps

Books the manuscript is not, to prevent mis-positioning.

The comp matrix must answer:

```md
What will an agent recognise?
What will an editor know how to sell?
What makes this manuscript different?
Why this author?
Why now?
```

Do not overclaim similarity to prize-winning books. Use comps to triangulate, not boast.

## Gate C — Agent List

Build a 40-agent target list.

No prestige guessing.

Each agent must qualify through at least two of these:

* represented literary memoir;
* sold memoir to serious imprints;
* represented African / diasporic literary nonfiction;
* represented faith-adjacent work without evangelical category reduction;
* represented family / inheritance / institutional-pressure nonfiction;
* states interest in literary memoir or narrative nonfiction;
* has acknowledged comparable books;
* has editor relationships at the target level.

For each agent record:

```md
Agent:
Agency:
Why this agent:
Relevant deals:
Relevant clients/books:
Submission requirements:
Requested materials:
Personalisation note:
Risk:
Priority tier:
```

## Gate D — Query Letter

The query must not sound like the book pleading its case.

It must do four things:

1. Open with the book's literary proposition.
2. State the central human pressure.
3. Position the manuscript accurately.
4. Establish author credibility without sounding defensive.

The query must not include:

* legal chronology;
* allegation detail;
* system critique;
* "this book will expose" language;
* "New York Times bestseller" language;
* "important story that needs to be told" language;
* therapy/recovery framing;
* father's-rights framing.

## Gate E — Synopsis

The synopsis must show narrative architecture, not list events.

It should track:

1. Hounslow present pressure.
2. Descent into origin.
3. Inheritance and covenant.
4. Loss and misread love.
5. Marriage and collapse.
6. Fatherhood under constraint.
7. Correction of love/fear.
8. Devotion inherited.
9. Final room / refusal to disappear.

The synopsis must not become:

* chronology;
* legal summary;
* spiritual proof;
* grievance record;
* family tree.

## Gate F — Sample Pages

Prepare:

* first 10 pages;
* first 25 pages;
* first 50 pages;
* full manuscript.

Each package must end on a line that makes the agent continue.

Do not assume agents will read past the requested sample.

Every sample boundary must be engineered.

## Gate G — Private Legal/Source Memo

Prepare a private, non-submitted internal memo for agent conversations.

It should list:

* legal sensitivities;
* protected material withheld;
* child-protection stance;
* pseudonym policy;
* sealed-material policy;
* what the manuscript deliberately does not say;
* what legal review is needed before publication.

This memo is not for query submission unless requested.

It protects the author and gives an eventual agent confidence that the manuscript is not reckless.

# Section 7 — Rejection-Prevention Stress Tests

Before querying, run these tests.

## Test 1 — The Unsympathetic Reader Test

Ask:

```md
Would this chapter still work for a reader who does not automatically believe Abraham?
```

If no, revise.

## Test 2 — The No-Case Test

Ask:

```md
Would this chapter still work if the reader knew nothing about the legal dispute?
```

If no, revise.

## Test 3 — The Secular Reader Test

Ask:

```md
Does the faith material remain credible to a reader who does not share Abraham's theology?
```

If no, revise.

## Test 4 — The Legal Hostility Test

Ask:

```md
Could any sentence be read as reckless, defamatory, procedurally unsafe, or child-sensitive?
```

If yes, review.

## Test 5 — The Page-30 Test

Ask:

```md
By page 30, has the reader encountered enough originality, authority, restraint, and pressure to continue?
```

If no, revise the first movement.

## Test 6 — The Middle-Book Trust Test

Ask:

```md
Does Part Three deepen trust, or does it ask for too much belief without rendering enough experience?
```

Ch.13 is the key chapter for this test.

## Test 7 — The Final-Image Test

Ask:

```md
Does Ch.22 feel inevitable, or merely correct?
```

Correct is not enough.

The ending must feel as though the book had been moving toward that room all along.

# Final Standard

The manuscript is not ready because it is moving.

The manuscript is ready only when every remaining objection has either been solved, intentionally held, or consciously accepted as unavoidable.

No avoidable weakness should reach an agent.

No private material should be used to make a public argument.

No child should carry the book.

No sentence should ask for sympathy when it can earn trust.

No chapter should explain what it has not rendered.

No submission should go out because the author is tired.

The book goes out when the strongest hostile reading cannot find an obvious reason to stop.

## WHAT YOU GOT RIGHT (That I Missed)

| My Mistake | Your Correction | Why You're Right |
|------------|----------------|------------------|
| One component to rule them all | Primitives + thin compositions | Single component becomes switch statement hell |
| Tokens as literal values | Tokens + semantic layer + CSS variables | Runtime flexibility without rebuilds |
| Delete first, ask questions later | Pilot surfaces, bridge mode, audit first | Legacy code has hidden business logic |
| Gradients as decoration | Gradients with scrims, ceilings, rules | Illegibility is the enemy of premium |
| Contrast as afterthought | 7:1/4.5:1 minimums, automated checks | Accessibility is not optional |
| Tailwind interpolation | CSS variables mapped to utilities | `bg-[${var}]` is not production-safe |

## NON-NEGOTIABLE RULES (To Be Enforced by Linting)

### Contrast Rule (Automated)

```typescript
// lib/design-system/contrast.ts
export const CONTRAST_MINIMUMS = {
  body: 7.0,      // WCAG AAA
  secondary: 4.5, // WCAG AA
  largeText: 4.5, // WCAG AA
  interactive: 4.5,
} as const;
```

### Gradient Rule (Code Review Required)

```markdown
No gradient may:
- Have opacity >12% beneath body text
- Use more than 2 color stops in text-bearing areas
- Allow text to sit directly on gradient without scrim
- Be animated (respect reduced-motion)
```

### Surface Contract Rule

```typescript
// Every surface must declare these properties
interface SurfaceContract {
  id: string;
  tone: 'warm' | 'neutral' | 'cool';
  density: 'spacious' | 'comfortable' | 'dense';
  heroScrim: 'none' | 'subtle' | 'pronounced';
  allowedGradients: GradientRecipe[];
  contrastTier: 'AAA' | 'AA';
}
```

### Ban List (ESLint Custom Rule)

```javascript
// Disallowed patterns
'bg-\\[\\${',           // No runtime Tailwind interpolation
'style=\\{\\{',          // No inline styles for colors
'#[0-9a-f]{3,6}',       // No raw hex codes outside tokens
'opacity-\\d+',         // No opacity without contrast validation (except transitions)
'gradient-to-[trbl]',   // No un-scrimmed gradients under text
```

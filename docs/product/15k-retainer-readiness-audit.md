## Direct answers

Can we sell £5k now?

Yes. Controlled retained oversight is operationally defensible.

Can we sell £15k now?

Almost, but only selectively. The product is now near-ready for executive oversight retainers where the client accepts a governed but still operator-led cadence.

What exactly blocks £15k?

- no scheduler-backed cadence
- conservative divergence memory rather than a dedicated multi-cycle divergence store
- operator roles are environment/config enforced, not schema-backed RBAC
- boardroom and counsel memory still rely on archive/audit persistence rather than dedicated long-lived models

What exactly remains before £50k?

- enterprise Control Room

- scheduler-backed cycle execution
- deeper boardroom history and counsel longitudinal outcome memory
- cross-organisation enterprise memory

What must not be claimed yet?

- full enterprise command centre
- automated cadence enforcement
- mature enterprise RBAC
- category-leading divergence memory

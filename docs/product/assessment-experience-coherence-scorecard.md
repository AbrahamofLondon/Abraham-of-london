# Assessment Experience Coherence Scorecard

> Date: 2026-05-08
> Criteria: Does each surface feel designed, not assembled?

---

## Scoring Dimensions (10 questions, each 0-10)

1. Does the opening frame prepare the user emotionally?
2. Does the question order move from accessible to difficult?
3. Does the user understand why the question matters?
4. Does the assessment avoid sounding like a survey?
5. Does it reward honesty?
6. Does it create a sense of being seen?
7. Does it preserve seriousness without becoming oppressive?
8. Does it produce a clear next step?
9. Does the next surface remember what was given?
10. Does the whole thing feel designed, not assembled?

## The Memory Problem

The weakest dimension across all surfaces is **"Does the next surface remember what was given?"** — average score: 4.0/10.

This is the core coherence defect. The product asks serious questions, extracts genuine evidence, but then each subsequent surface starts from near-zero context. The user gives deeply personal or structurally consequential answers, and the next stage acts as if it never heard them.

**This is not a question-quality problem. It is an architecture problem.** The questions are mostly strong. The memory is mostly broken.

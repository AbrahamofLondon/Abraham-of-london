# Boardroom Archive Standard

Boardroom memory is part of retainer lock-in.

## Archive Minimum

- case id
- cycle id
- qualification timestamp
- dossier summary
- trigger reason
- cost threshold where real
- recommended decision path
- objections count if available
- export status

## Product Use

- Oversight brief may show previous dossier count, unresolved board-level issues, and repeated board-level exposure.
- Decision Centre may show historical boardroom count per case.
- Operator bench may use archive memory when deciding whether the same board-grade issue is recurring.

# Constitutional Question Quality Scorecard (Deep Dive)

> Date: 2026-05-08
> Rubric: 10 dimensions, each 1-10. Total /100. Minimum bar: 70.
> Context: Scored after deep downstream analysis, not surface impression.

---

## Dimensions

1. **Specificity** — can the user answer concretely, not abstractly?
2. **Evidence yield** — does the answer produce usable diagnostic signal?
3. **Emotional pressure** — does it create useful discomfort?
4. **Authority signal** — does it test decision authority?
5. **Contradiction signal** — can it expose conflicting truths?
6. **Recurrence signal** — does it test whether this has happened before?
7. **Governance value** — does it feed posture/readiness/discipline classifications?
8. **Route-decision value** — does it affect REJECT/DIAGNOSTIC/STRATEGY?
9. **Downstream usefulness** — does it improve Strategy Room, Executive Reporting, Return Brief, Oversight?
10. **Resistance to generic interpretation** — would a competitor product ask this?

## Position

No question requires replacement. The 4 elite questions (q1, q2, q4, q7) define the assessment's category. The 2 rewritten questions (q5, q9) earn their place. The 2 adequate questions (q3, q6) are functional. The 2 weakest (q8, q10) serve necessary roles that no other question covers.

The Constitutional Diagnostic is not a question-quality problem. It is a bridge-quality problem. The questions extract the right signals. The bridge transmits those signals as numbers instead of as evidence.

# Counsel History Standard

Counsel is governed escalation, not consultancy.

## Rules

- Every counsel trigger must be recorded as a workflow event.
- Every counsel interaction must become structured evidence or structured refusal.
- Informal side-channel notes do not count as governed counsel history.
- Client-safe sponsor output suppresses sensitive counsel detail.
- Board-safe output may preserve consequence-level counsel posture without exposing private notes.

## Minimum History Fields

- trigger reason
- trigger source
- workflow status
- assignment intent
- requested review question
- recommendation summary
- evidence nodes created or referenced
- operator acceptance / rejection / pending disposition
- resulting action

# Counsel Workflow Operational Standard

Counsel is governed evidence. Not opinion theatre.

## Workflow states

1. Assignment.
2. Review acceptance.
3. Review submission.
4. Recommendation.
5. Evidence node generation.
6. Decision-log effect.
7. Outcome pending marker.
8. Counsel effectiveness tracking.
9. Escalation to retained oversight.
10. Escalation to boardroom where required.

## Required review output

- Review status.
- Trigger reason.
- Evidence basis.
- Recommendation.
- Contradiction assessment.
- Risk if ignored.
- Whether counsel agrees with the system restriction.
- Required client action.
- Outcome follow-up date.

## Prohibited

- Counsel bypassing governance.
- Counsel replacing evidence.
- Counsel producing ungated client-facing raw commentary.

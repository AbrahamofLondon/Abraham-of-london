# Executive Reporting Compression Map

## Before

The result page opened with a broad spread of intelligence blocks:

1. Board snapshot
2. Evidence carried forward
3. Boardroom dossier anchor
4. Opening cost line
5. Projected cost of inaction
6. AI-adjusted consequence projection
7. Identified condition
8. Contradiction
9. Consequence model
10. Decision required
11. Required actions
12. If ignored
13. Competitive position / terrain / predictive consequence
14. Required move
15. Advantage path
16. Ladder progression gate
17. Longitudinal / divergence / outcome verification
18. Evidence convergence and closing method notes

The issue was not missing data. The issue was comprehension order.

## After

The result hierarchy is compressed into this top sequence:

1. Dominant finding
2. Governed priority stack
3. Required decision
4. Financial exposure / cost of delay
5. Boardroom readiness
6. Evidence carried forward
7. Next admitted step

Everything else is treated as supporting material under:

- Evidence, governance, and method

## Mapping

- Dominant finding
  Uses the identified condition, contradiction, and consequence headline.
- Governed priority stack
  Uses the governed required actions / priority stack already produced by the report.
- Required decision
  Uses the decision object and required move copy.
- Financial exposure / cost of delay
  Uses the opening cost line, projected cost of inaction, consequence model, and irreversibility estimate where evidence is sufficient.
- Boardroom readiness
  Uses boardroom qualification state and dossier availability.
- Evidence carried forward
  Uses inherited governance evidence already present in the report.
- Next admitted step
  Uses the Strategy Room progression gate and checkpoint-aware execution path.
- Evidence, governance, and method
  Retains board snapshot, dossier detail, terrain/AI blocks, comparative intelligence, longitudinal panels, divergence, outcome verification, and evidence convergence notes without deleting substance.

## Compression rule

No data was deleted for this pass. The change is hierarchy, not subtraction.

# Institutional Memory Archive Standard

The system cannot keep pretending each cycle starts from scratch.

## The archive must preserve

1. Decisions under governance.
2. Recurring contradictions.
3. Unresolved patterns.
4. Intervention attempts.
5. Commitments.
6. Breaches.
7. Verified outcomes.
8. Cost history.
9. Irreversibility movement.
10. Boardroom history.
11. Counsel history.
12. Sponsor actions.
13. Recurrence events.
14. Cadence failures.
15. Value protected.
16. Visibility lost if oversight stops.

## The archive must produce

- Memory summary.
- Pattern history.
- Intervention history.
- Boardroom history.
- Counsel history.
- Cost history.
- Irreversibility history.
- Decision credit movement.
- Visibility lost if cancelled.

## Truth standard

If the archive cannot prove a count from persisted history, it does not appear in cancellation-loss language.

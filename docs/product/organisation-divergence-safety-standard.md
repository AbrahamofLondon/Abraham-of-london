# Organisation Divergence Safety Standard

Organisation divergence must remain aggregation-first.

## Allowed

- sponsor-safe summaries
- affected domain
- divergence type
- respondent count
- confidence
- suppressed detail count
- recommended next action

## Suppressed

- raw respondent text
- anonymous identities
- small-sample slices
- respondent-level psychological framing

## Safe Types

- authority perception gap
- blocker contradiction
- cost estimate divergence
- condition class mismatch
- execution confidence gap
- strategic priority split
- evidence quality gap

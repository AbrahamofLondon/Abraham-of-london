# Portfolio Command Standard

The portfolio layer exists to prevent a fake enterprise story built from a single-organisation retainer with premium pricing.

## Rules

- Cross-organisation intelligence is anonymised by default.
- Named comparison requires explicit portfolio entitlement.
- No client-to-client leakage.
- No raw case text.
- No respondent identity.
- No public proof claims sourced from private portfolio evidence.

## Required outputs

- Shared contradiction patterns.
- Leadership failure mode spread.
- Fastest deterioration across organisations.
- Board-level exposure concentration.
- Overdue sponsor intervention concentration.
- Recurrent counsel escalation concentration.
- Decision credit improvement pockets.
- Spreading governance risk.

## Status language

- `NO_SHARED_PATTERN`
- `POSSIBLE_PORTFOLIO_PATTERN`
- `VERIFIED_PORTFOLIO_PATTERN`
- `ESCALATING_PORTFOLIO_PATTERN`
- `SUPPRESSED_PRIVACY_RISK`
- `INSUFFICIENT_SAMPLE`

## Hostile truth

Until explicit entitlement and harder multi-tenant boundaries exist, this layer stays behind the curtain.

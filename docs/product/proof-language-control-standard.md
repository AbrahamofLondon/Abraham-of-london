# Proof Language Control Standard

**Date:** 2026-05-08
**Authority:** Decision Infrastructure by Abraham of London

---

## Forbidden Proof Language on Public Surfaces

| Term | Why | Replace With |
|------|-----|-------------|
| "proven" | Implies scientific proof without methodology | "evidence-supported" or "outcome-supported" |
| "guaranteed" | No outcome can be guaranteed | Remove or rephrase |
| "verified" (without method) | Implies external verification | "verified through [method]" |
| "statistical" (without N) | Implies statistical rigour without sample | "observed in N cases" |
| "outcome" (without classification) | Implies verified outcome | "outcome (self-reported)" or "outcome (operator-confirmed)" |
| "AI-powered proof" | Marketing language, not institutional | "governed analysis" |
| "platinum" (unless issuable) | PLATINUM is not currently issuable | "reserved until evidence threshold is met" |
| "case study" (unless sealed) | Implies published, reviewed evidence | "anonymised case pattern" |
| "live evidence" (if fallback) | Misleads about data source | "demonstration pattern" |
| "behaviour verified" (if self-reported) | Self-report is not verification | "user-reported outcome" |
| "institutional intelligence" (if case-level) | Overstates scope | "case-level evidence" |
| "success rate" (without denominator) | Meaningless without methodology | "observed improvement in N/M cases" |
| "scientifically proven" | We are not a scientific institution | Remove |
| "deterministic demonstration" | Technical jargon | "governance pattern demonstration" |

## Preferred Vocabulary

- evidence-supported
- publication-eligible
- review-gated
- anonymised aggregate
- demonstration pattern
- verified within the review window
- outcome-supported
- operator-confirmed
- documentary trace
- client-safe
- insufficient sample
- reserved until evidence threshold is met
- observed in N cases
- associated with (not "caused by")

## First Encounter Proof Language

### Instead of:
"This is a deterministic demonstration. Real decisions are scored by the C3 fidelity engine, contradiction kernel, action simulation, and outcome verification system."

### Use:
"This demonstration shows the governance pattern. Real decisions are evaluated against clarity, contradiction, consequence, authority, and follow-through before deeper access is granted."

This protects IP and sounds more professional.

## Value Protected Language

### Instead of:
"Cancel and you lose everything."

### Use:
"Continued oversight is preserving visibility across: recurring decision patterns, unresolved commitments, accumulated cost exposure, and board-level escalation history."

Each claim tied to a real data category with evidence labels:
- based on archived cycles
- based on active cases
- based on verified commitments
- based on boardroom dossier history
- based on sponsor-safe organisation data

## Key Findings

### High-Yield Questions to Protect (Never Remove or Weaken)
1. Fast Diagnostic Q1 (decision) — anchors entire case
2. Fast Diagnostic Q2 (authority) — irreplaceable authority signal
3. Fast Diagnostic Q3 (consequence) — cost anchor
4. Fast Diagnostic Q4 (commitment) — unique commitment gate
5. Purpose Alignment: identity_1, identity_2, identity_3, decision_1, decision_3
6. Constitutional: q1, q2, q4, q7, q8
7. All adaptive question bank templates
8. All return brief prompts

### Low-Yield Questions to Remove
1. environment_3 (score 11) — media consumption
2. legacy_1 (score 12) — aspirational platitude
3. legacy_2 (score 12) — vague
4. All 5 enterprise respondent questions (avg 11.4)
5. Confidence baseline slider (team post-reflection)

### Questions That Feel Important But Produce Weak Evidence
- environment_1 ("five people making me sharper") — memorable but only scores 15. It creates emotional impact without producing decision intelligence.
- legacy_3 ("taking on harder things") — confrontational but only scores 16. It challenges comfort-seeking without producing actionable data.
- eco_2 ("disagree openly") — socially important but produces social desirability bias, not truth.

### Questions That Produce Strong Evidence But Are Badly Worded
- q5 ("trust materially intact") — scores 23 despite confusing wording. With clearer language, would score higher.
- q9 ("structural not motivational") — scores 27 despite requiring expert-level causal reasoning. Simpler wording would improve response quality.
- "Execution reality matches what leadership believes" — strong contradiction yield (respondent view) but meta-cognitive framing makes it hard to answer.

### Missing Questions That Would Materially Improve Downstream Outputs
1. **VERIFICATION:** "What evidence would prove this intervention succeeded?" — missing from every surface.
2. **RECURRENCE:** "Has this exact problem occurred before in a different role/organisation?" — would massively improve pattern engine.
3. **PRIVACY_BOUNDARY:** "Is there information here that should not be shared beyond this system?" — would improve data handling.
4. **STAKEHOLDER_RISK (direct):** "Who will be affected most by this decision, and do they know?" — would improve boardroom usefulness.
5. **COMMITMENT (mid-ladder):** "Based on what you've seen so far, what is the first thing you will change?" — missing between Fast Diagnostic and Strategy Room.

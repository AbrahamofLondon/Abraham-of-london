# Retainer Indispensability Standard

The brief should show what visibility would be lost if oversight stopped.

## Tone

Do not say:

- don’t cancel
- you will fail without us
- you must retain us

Say:

- If oversight stopped today, the following visibility would no longer be maintained.
- Without continued oversight, this pattern may become harder to detect early.
- The following signals would likely return to manual tracking or be lost between reviews.

## Evidence Basis

Indispensability may refer only to signals the system is actually preserving:

- cost visibility
- recurrence detection
- commitment verification
- boardroom archive memory
- counsel history
- organisation divergence
- cycle comparison
- archived governed decisions

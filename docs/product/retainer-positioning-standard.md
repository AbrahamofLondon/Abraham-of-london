# Retainer Positioning Standard

## Allowed language

- retained oversight
- sponsor-safe command visibility
- institutional memory
- oversight cadence
- counsel escalation history
- boardroom archive
- evidence retained across cycles
- active continuity

## Avoid

- fully automated
- enterprise intelligence platform
- AI-powered oversight
- guaranteed improvement
- verified transformation
- £50k retainer ready
- proprietary engine
- kernel
- trigger mechanics

## Pricing posture

- £5k: defensible for retained oversight foundation.
- £15k: defensible selectively with controlled delivery and operator involvement.
- £50k: not public, not standard, only bespoke if manually justified and operator-led.

# Visibility Retained Language Standard

This panel exists to state consequences of lost oversight without sounding needy, manipulative, or false.

## Allowed framing

- "Visibility would reduce across..."
- "Continued oversight preserves..."
- "Current oversight is retaining..."
- "Insufficient evidence to state..."

## Forbidden framing

- "You will lose everything."
- "Act now or this disappears."
- "Only we can preserve this."
- Any sales language.
- Any fear language unsupported by persisted evidence.

## Evidence rule

Only show categories backed by real data in the current case or archive.

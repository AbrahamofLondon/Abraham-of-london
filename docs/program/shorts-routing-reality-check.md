# Shorts Routing Reality Check

Status: PARTIAL
Live surface truth: The homepage Shorts rail currently supports three practical intents only: diagnostics, intelligence, and Strategy Room escalation.

## Objective

Ensure the Shorts/content routing language does not claim broader routing capability than the current site actually provides.

## Live Routing Now

| Short title | Intent | Live CTA |
|---|---|---|
| You Don't Have a Growth Problem | diagnostic | `/diagnostics/constitutional-diagnostic` |
| Your Leadership Team Isn't Aligned | diagnostic | `/diagnostics/constitutional-diagnostic` |
| You Are the Bottleneck | diagnostic | `/diagnostics/constitutional-diagnostic` |
| Nothing Looks Broken | intelligence | `/artifacts` |
| This Isn't a Content Problem | strategy room | `/strategy-room` |

## What This Means

- `Shorts -> Diagnostics` is live
- `Shorts -> Intelligence` is live, but currently uses `/artifacts` as the practical destination
- `Shorts -> Strategy Room` is live

## What Is Not Proven Yet

- broader per-short audience segmentation
- dynamic personalized routing
- a dedicated intelligence index replacing `/artifacts`
- richer multi-step content-routing automation

## Doc Correction

Authority/content docs should describe Shorts as a curated signal layer that can route into:

- diagnostics
- intelligence
- strategy escalation

They should not imply a broader automated routing system than currently exists.

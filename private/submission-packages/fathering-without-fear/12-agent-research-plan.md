# Agent Research Plan — Fathering Without Fear

## Purpose

Identify literary agents currently open to queries in literary memoir and literary nonfiction who are a strong fit for *Fathering Without Fear*. Do not create an agent list yet. This plan defines the research method and scoring rubric.

## Research Method

For each potential agent:

1. Visit the agency's current website or verified submission portal
2. Confirm the agent is currently active and accepting queries
3. Confirm the agent represents memoir / literary nonfiction
4. Review the agent's manuscript wish-list or recent deals for fit indicators:
   - Interest in formally compressed or short-form nonfiction
   - Interest in international / Nigerian / British diaspora narratives
   - Interest in fatherhood, family, inheritance, or grief as literary subjects
   - Tolerance for faith as lived experience without requiring devotional genre
5. Review submission guidelines for required materials
6. Record source URL and date checked
7. Score against rubric
8. Assign priority

## Scoring Rubric

| Criterion | Score | Description |
|---|---|---|
| **Literary memoir fit** | 0–5 | Does the agent actively represent literary memoir? 0 = no, 5 = primary category |
| **Compressed/formal nonfiction openness** | 0–5 | Is the agent open to short or fragmentary nonfiction? 0 = requires conventional length, 5 = actively seeks compressed work |
| **International/Nigerian/British context fit** | 0–5 | Does the agent represent diaspora or cross-cultural narratives? 0 = no interest, 5 = actively seeks |
| **Faith-inflected but non-devotional tolerance** | 0–5 | Can the agent position faith as lived experience without requiring devotional genre? 0 = faith is a problem, 5 = comfortable with literary faith |
| **Fatherhood/family/grief/inheritance interest** | 0–5 | Does the agent represent books on these themes? 0 = no, 5 = actively seeking |
| **Submission openness/current status** | 0–5 | Is the agent currently open to queries? 0 = closed, 5 = open and responsive |
| **Overall priority** | A/B/C/Reject | A = strong fit, query first. B = possible fit, query after A list. C = marginal fit, query only if A/B exhausted. Reject = does not fit criteria. |

## Scoring Guidelines

- **A (Priority):** Score ≥ 20 across all criteria, with no single criterion below 3. Strong thematic and formal fit. Query first.
- **B (Secondary):** Score 15–19, or strong fit in most criteria but one area of weakness. Query after A list.
- **C (Tertiary):** Score 10–14, or marginal fit. Query only if A/B lists are exhausted.
- **Reject:** Score < 10, or any disqualifying factor (e.g., only represents celebrity memoir, requires platform, no memoir category).

## Disqualifying Factors

Reject immediately if the agent:

- Represents only celebrity memoir or platform-driven nonfiction
- Requires a large social media following or established public profile
- Specialises exclusively in trauma memoir or "survivor narrative"
- Seeks prescriptive parenting or self-help nonfiction
- Would position the book as legal exposé or campaign journalism
- Is uncomfortable with faith as a presence in literary nonfiction
- Would require the manuscript to be expanded to 60,000+ words before submission
- Is not currently open to queries
- Has no track record in literary memoir or literary nonfiction

## Research Sources

- Agency websites (verified, not aggregated directories)
- Publishers Marketplace deals database
- Literary agency submission guidelines pages
- Manuscript wish-list posts (Twitter/X, Bluesky, agency blogs)
- Recent (within 12 months) interviews with agents about what they're seeking
- Author acknowledgements in similar memoirs (who represented the author)

## Required Verification

Each agent on the final list must be confirmed through:

- Current agency website or verified submission guidelines — source URL and date checked recorded
- Confirmation the agent is currently open to unsolicited queries
- Confirmation the agent represents memoir / literary nonfiction
- No disqualifying factors present
- Score recorded with rationale

## Output

The final agent list will be recorded in a separate file with:

- Agent name
- Agency
- Source URL and date checked
- Score per criterion
- Overall priority (A/B/C/Reject)
- Submission materials required
- Notes

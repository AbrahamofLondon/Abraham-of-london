# Agent Submission Strategy
## The Fiction Adaptation — Staged Query Plan

**Strategy Type:** Tiered, staged submission with tracking  
**Total Target:** 15+ agents across 3 tiers  
**Submission Timeline:** 3 months minimum (longer with care)  
**Primary Goal:** Secure agent representation for literary speculative fiction  
**Secondary Goal:** Gather feedback for manuscript refinement if needed

## WAIT PERIODS & FOLLOW-UP PROTOCOL

### Standard Wait Period: 6–8 Weeks

**Why 6–8 weeks?**
- Standard industry guideline for literary agents
- Some agents state "up to 12 weeks"; most respond within 8
- Earlier follow-up risks looking impatient; later looks forgotten

**Week 4–5 Checkpoint:**
- No response yet is normal; do not follow up yet
- Use time to refine positioning for Phase 2 if Phase 1 yields feedback

**Week 6–8 Decision Point:**
- Gather emails from non-responsive agents
- Prepare follow-up emails for those who don't specify no-response-means-no

**Follow-up Protocol:**
- Check agency guidelines: Does "no response means no" apply?
- If no explicit policy, send ONE follow-up email at 8 weeks
- Follow-up subject line: "RE: Query — The Fiction Adaptation (Follow-up)"
- Follow-up message: Brief, professional, just restates availability of manuscript
- Do not send multiple follow-ups to same agent

**After 8 Weeks + Follow-up:**
- No response = Consider as pass
- Move forward with next tier
- Do not revisit same agent unless they request

## RESPONSE TRACKING & ANALYSIS

### Response Categories

**1. Full Manuscript Request**
- Status: PRIORITY
- Action: Respond within 24 hours with materials in requested format
- Note: This is strongest possible positive signal
- Pause lower-tier submissions if multiple requests to manage agent relationships

**2. Partial Manuscript Request** (sample pages, chapters)
- Status: POSITIVE
- Action: Respond within 24 hours
- Note: Manuscript has caught their interest; they're vetting fit

**3. Rejection with Feedback**
- Status: VALUABLE DATA
- Action: Read feedback carefully; analyze if it's about fit or execution
- Note: Use to refine query for remaining agents if feedback is actionable
- Do not argue or respond to feedback; simply learn and move forward

**4. Rejection without Feedback** (form rejection)
- Status: EXPECTED (most common)
- Action: Record and move forward
- Note: Not personal; not unusual for unsolicited queries
- Do not seek explanation

**5. No Response at 8 Weeks + Follow-up**
- Status: PASS (treat as rejection)
- Action: Record and move forward
- Note: Standard outcome for many agents; expected
- Do not send additional emails

### Feedback Analysis Template

If rejections include feedback, analyze:

| Feedback Pattern | Possible Meaning | Action |
|---|---|---|
| "Not right for my list" | Positioning or niche mismatch | Review query; consider different agent tier |
| "Unsure about commercial viability" | Market positioning unclear | Strengthen comp line rationale; verify Tier 1 alignment |
| "Compelling writing but..." | Execution is strong, story/fit is issue | Query is solid; move to next agent |
| "Something special but..." | Quality is recognized, agent not right fit | Indicates manuscript has strength; continue to next agents |
| No feedback | Most common outcome | Expected; do not over-interpret |

## CONTINGENCY SCENARIOS

### Scenario 1: One or More Tier 1 Agents Request Full Manuscript

**Immediate Action:**
- Respond within 24 hours with materials in requested format
- Pause all Tier 2 submissions
- Notify other pending Tier 1 agents: "My manuscript is currently under review. I will follow up if/when it becomes available again."
- Continue waiting for Tier 1 responses (do not withdraw manuscript)

**Timeline:**
- Agents typically read full manuscript in 4–8 weeks
- Most respond within 12 weeks
- If no response by 12 weeks, send polite follow-up

**Expected Outcome:**
- If agent passes: Resume Tier 2 submissions
- If agent offers representation: Excellent; consider pausing all other queries while negotiating
- If agent wants revisions: Evaluate feedback; consider if revisions strengthen manuscript for other agents

### Scenario 2: No Requests from Tier 1 After 8 Weeks

**Action:**
- Send follow-up emails to Tier 1 agents who don't respond
- Analyze any feedback from rejections
- Proceed to Phase 2 (Tier 2 submissions)
- Use Phase 1 feedback to refine query if patterns emerge

**Possible Refinements:**
- Strengthen comp line rationale
- Adjust pitch emphasis
- Clarify genre positioning
- Verify personalisation accuracy

### Scenario 3: Multiple Requests from Tier 1 & 2

**Excellent Scenario**
- Manage multiple manuscript review processes simultaneously
- Respond promptly to all requests
- Prepare for possible competing offers
- Consider pausing Phase 3 to manage existing agent interest
- If offers come: Evaluate agents and negotiate (or consult literary lawyer if needed)

### Scenario 4: Pattern of Rejection Despite Strong Manuscript

**Analysis:**
- If feedback suggests quality is strong but "not right for their list": Continue to next agents (not uncommon)
- If feedback suggests market positioning issue: Consult with trusted readers; consider query refinement
- If feedback is consistently about genre misalignment: Review agent targeting; verify Tier 1 selections

**Action:**
- Continue through Tier 2 and 3
- Do not assume manuscript is weak; agent fit is often the issue
- Manuscript quality and agent fit are separate matters

## PROFESSIONAL CONDUCT DURING SUBMISSIONS

### DO

✅ Keep queries personalized and professional  
✅ Follow submission guidelines exactly  
✅ Respond promptly to requests or rejections  
✅ Track all submissions meticulously  
✅ Accept rejection professionally  
✅ Maintain professional tone in all communications  
✅ Respect agent timelines and decision processes  

### DON'T

❌ Send to multiple agents at same agency  
❌ Withdraw manuscript without notice  
❌ Argue with or question rejections  
❌ Resend query if rejected  
❌ Follow up more than once  
❌ Send unsolicited full manuscript  
❌ Query agents without reading guidelines  
❌ Invent personalisation details

## SUCCESS CRITERIA

**Submission is successful if:**

1. **Best case:** Agent offers representation (obviously)
2. **Good case:** Multiple agents request full manuscript
3. **Solid case:** At least one agent from Tier 1 or 2 requests manuscript
4. **Acceptable case:** Reach Tier 3 submissions; continue gathering responses
5. **Learning opportunity:** Even if no offers, gather feedback and market intelligence for manuscript strength

**Do not consider submission unsuccessful until:**
- All 15+ agents have responded
- Minimum 12 weeks have passed since last Phase 3 submission
- No outstanding manuscript requests remain

## TIMELINE OVERVIEW

**Month 1:**
- Week 1: Send Tier 1 batch (5–7 agents)
- Weeks 2–4: Wait and monitor

**Month 2:**
- Weeks 5–6: Send Tier 2 batch (5–7 agents)
- Week 7–8: Follow up with any non-responsive Tier 1 agents

**Month 3:**
- Weeks 9–10: Send Tier 3 batch (5–8 agents)
- Weeks 11–12: Wait for final responses from Phase 2
- Weeks 12+: Continue waiting for Phase 3 responses

**Contingency:**
- If manuscript request arrives: Pause tiers; manage manuscript review
- If offer arrives: Pause all submissions; negotiate with agent
- If pattern emerges requiring query refinement: Stop Phase 3; refine and reconsider Phase 2

Before sending any query, verify:

**Query Letter:**
- [ ] Author-positioning line is filled in
- [ ] Contact details are correct and professional
- [ ] Agent name is spelled correctly
- [ ] Comp line is correct (Dept. of Speculation + Our Wives Under the Sea)
- [ ] Word count is correct (100,644 words)
- [ ] Genre is correct (literary speculative fiction)
- [ ] Personalisation hook is factual and specific
- [ ] Professional closing with availability statement
- [ ] No typos or formatting errors

**Synopses:**
- [ ] One-page synopsis is provided (if requested)
- [ ] Three-page synopsis is provided (if requested)
- [ ] Format matches agent guidelines (font, spacing, length)
- [ ] Ending is disclosed (agents expect this)
- [ ] No spoiler minimization

**First Pages (if requested):**
- [ ] First 10–25 pages (per agent guidelines)
- [ ] Professional formatting (1-inch margins, standard font)
- [ ] No headers/footers unless specified
- [ ] Clean, no tracked changes

**Submission Method:**
- [ ] Follow agency submission guidelines exactly
- [ ] Email subject line follows format specified
- [ ] Attachments in requested format (Word, PDF, inline text)
- [ ] Email from professional address
- [ ] Send at professional hour (morning business hours, your timezone)

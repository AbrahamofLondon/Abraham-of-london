## TIER 1: Highest Alignment (5–7 Agents)

### Agent profiles to research and verify:

Tier 1 agents are those with:
- Active representation of literary speculative fiction or literary fiction with metaphysical elements
- Recent clients in marriage-centered or domestic literary fiction
- Explicit interest in restrained prose, ambiguity, voice-led work
- Published deals in similar positioning to The Fiction Adaptation

**Research checklist for each Tier 1 agent:**
- [ ] Agent name, agency name, agency location
- [ ] Current submission status (open/closed/selective)
- [ ] Submission method (email/form/QueryTracker/other)
- [ ] Recent 3 clients in relevant category
- [ ] Recent deal (2024–2026) in similar positioning
- [ ] #MSWL post or interview quote (if available)
- [ ] Agency website link
- [ ] Submission guidelines link
- [ ] Date verified

**Research framework for Tier 1:**

**1. Agents who represent the comps:**
- Find: Agent for Jenny Offill (*Dept. of Speculation*)
  - Source: Acknowledgements page, Publishers Marketplace, agent bio
  - Why: Direct representation of primary comp
  
- Find: Agent for Sarah Read or Catherine Armfield (related to *Our Wives Under the Sea*)
  - Source: Pan Macmillan author page, acknowledgements, Publishers Marketplace
  - Why: Direct representation of primary comp

**2. Literary agents actively seeking metaphysical/speculative literary fiction:**
- Search QueryTracker: "literary speculative," "magical realism," "domestic metaphysical," "literary supernatural"
- Search MSWL: #MSWL posts mentioning these terms
- Look for agents on these publishers: Knopf, FSG, Norton, Pan Macmillan, Granta Books

**3. Agents with recent clients in similar category:**
- Verify recent deals (2024–2026) in literary fiction + metaphysical/supernatural elements
- Verify recent deals in marriage-centered literary fiction
- Look for restrained prose, voice-led positioning in client books

## NOTES FOR RESEARCH

- **Do not invent agent information.** If submission status is unclear, mark as "Verify current status on agency website before sending."
- **Document all sources.** Each agent entry must have links and dates.
- **Verify current MSWL.** Agent interests can change; use recent posts/statements only.
- **Check submission guidelines weekly.** Agents frequently change status or guidelines; verify again before sending each query.
- **Prioritize recent clients.** Look for sales in 2024–2026 for current evidence of fit.
- **Look for voice and positioning.** Not just genre, but specific editorial interests in restrained prose, ambiguity, character depth.

## Sending Protocol

### Before Sending Any Tier 1 Query

1. **Verify submission guidelines** for that agent (within 24 hours of sending)
2. **Check AI-policy status** — see ai-policy-review.md
3. **Check same-agency rule** — confirm no simultaneous submissions to same agency if prohibited
4. **Complete pre-send checklist** (see pre-send-owner-checklist.md)
5. **Owner approves** — owner manually confirms "okay to send" for each query
6. **Send query** — owner sends from info@abrahamoflondon.org
7. **Update tracker** — enter send date, response deadline, follow-up date

### Response Tracking

**When response arrives:**
- Enter response type in "Response" column
- Update "Request Status" with outcome
- Add any relevant notes
- If full manuscript requested: Prepare manuscript in agent-requested format; send within 24 hours
- If pass: Record feedback if provided; move to next agent
- If no response by deadline: Send follow-up email at 8-week mark; update "Follow-Up Date"; move tracker to "Follow-up sent" status

### Follow-Up Protocol

**If no response after 8 weeks:**
1. Check agency guidelines: Do they state "no response means no"?
2. If no explicit policy stated, send ONE polite follow-up email
3. Subject line: "RE: Query — The Fiction Adaptation (Follow-up)"
4. Body: Brief, professional; simply restate manuscript availability
5. Do NOT send multiple follow-ups
6. If still no response after follow-up, mark as "no response" and move to next agent

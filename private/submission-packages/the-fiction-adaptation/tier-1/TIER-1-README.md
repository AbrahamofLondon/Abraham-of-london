## Folder Structure

Each agent has two folders:

### Planning Folder (e.g., `01-olivia-maidment/`)

Contains all preparation and guidance for sending to that agent:

- **materials-to-send.md** — What to attach, email format, submission method
- **submission-checklist.md** — Verification before send
- **submission-notes.md** — Special considerations or agent-specific guidance
- **personalised-query.md** — The tailored query letter
- **guideline-verification.md** — Record of guidelines checked and when
- **ai-policy-note.md** — AI-policy status for this agent

### Export Folder (e.g., `exports/01-olivia-maidment/`)

Contains production-ready materials for sending:

- **export-readiness-report.md** — Confirmation that materials are production-ready (or hold reasons if not)
- **metadata-scrub-report.md** — Verification that DOCX files are clean
- **query-letter-source.md** — Markdown source of personalized query
- **synopsis-source.md** — Markdown source of synopses
- **sample-pages-source.md** — Markdown source of sample pages (if needed)
- **DOCX files** (if appropriate to this agent's guidelines)
  - The-Fiction-Adaptation_Query_[AgentName].docx
  - The-Fiction-Adaptation_1pSynopsis_[AgentName].docx
  - The-Fiction-Adaptation_3pSynopsis_[AgentName].docx
  - The-Fiction-Adaptation_SamplePages_[AgentName].docx (if applicable)

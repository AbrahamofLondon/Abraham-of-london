## Instructions for Updating

### When Sending a Query

**Enter:**
- **Guidelines:** Date checked (MM/DD/YYYY)
- **Sent:** Date sent (MM/DD/YYYY)
- **Deadline:** Expected response date (typically 8 weeks from send)
- **Follow-up:** Calculated follow-up date (8 weeks from send)

### When Receiving a Response

**Enter:**
- **Response:** Date response received (MM/DD/YYYY) or "No response at deadline"
- **Type:** Response type from this list:
  - Full MS request (full manuscript requested)
  - Partial request (50 pages or first chapters requested)
  - Pass with feedback (rejection with explanation)
  - Pass (form rejection, no explanation)
  - No response (deadline exceeded, no response)
  - Follow-up sent (sent one follow-up email at 8-week mark)

### In Notes Column

**Record:**
- Personalised rejection feedback (if provided)
- Special context ("requested manuscript; reading now")
- Agent milestones ("requested full MS; got it, now waiting")
- Timeline notes

## Response Categories & Next Steps

### Full Manuscript Request
- ✅ **Priority signal** — Agent wants to read entire book
- Action: Send full manuscript within 24 hours
- Format: Check agent guidelines for preferred format (Word, PDF, etc.)
- Note: Pause lower-tier submissions if managing multiple requests

### Partial/Sample Request
- ✅ **Positive signal** — Agent is testing fit
- Action: Send requested pages within 24 hours
- Note: Follow agent-specified format exactly

### Pass with Feedback
- **Valuable data** — Agent explains reason for pass
- Action: Record feedback carefully
- Analysis: If pattern emerges (e.g., multiple agents: "not right for list"), consider query refinement
- Note: Do not respond to rejection; simply learn and move on

### Pass (Form Rejection)
- **Expected outcome** — Standard rejection, no explanation
- Action: Record; move to next agent
- Note: Not personal; common for unsolicited queries

### No Response at Deadline
- **Standard outcome** — Agent did not respond within 8 weeks
- Action: Send ONE polite follow-up email at 8-week mark
- Follow-up subject: "RE: Query — The Fiction Adaptation (Follow-up)"
- Follow-up body: 2–3 sentences, professional, restate manuscript availability
- Note: Do not send multiple follow-ups

### Follow-Up Sent
- **Tracking status** — One follow-up email has been sent
- Action: Wait 2 more weeks; if still no response, mark as final pass
- Note: Do not send second follow-up
